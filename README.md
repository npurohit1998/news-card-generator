# फलोदी News — Card Studio PWA

## फ़ाइलें
- `index.html` — मुख्य ऐप
- `manifest.json` — PWA manifest
- `sw.js` — Service Worker (offline support)

## Deploy कैसे करें (Free)

### Option 1: GitHub Pages (सबसे आसान)
1. GitHub पर free account बनाएं: github.com
2. New Repository बनाएं: `phalodi-news`
3. तीनों files upload करें
4. Settings → Pages → Branch: main → Save
5. URL मिलेगा: `https://[username].github.io/phalodi-news`

### Option 2: Netlify (drag & drop)
1. netlify.com पर free account
2. इस folder को drag & drop करें
3. तुरंत live URL मिलेगा

### Option 3: Vercel
1. vercel.com पर free account
2. GitHub से connect करें
3. Auto-deploy होगा

## PWA Install करना (Mobile पर)
- Chrome में खोलें → Menu → "Add to Home Screen"
- iOS Safari → Share → "Add to Home Screen"

## Icons बनाना
- `icon-192.png` (192×192 px) और `icon-512.png` (512×512 px)
- canva.com पर free में बना सकते हैं
- Logo वाला icon upload करें
